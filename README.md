# Accounting Ledger

A comand line Java app for managing and tracking financial records.

## Features

- Add deposits and payments
- View full ledger w/ filter options
- Generate reports:
  - Month to date
  - Previous month
  - Year to date
  - Previous year
  - Search by vendor
